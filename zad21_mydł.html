Jakie jest znaczenie znaku # znak # służy do Wywołania identyfikatora
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie identyfikatora to id="nazwa zdefiniowanego w CSS identyfikatora" czyli:
Odpowiedni wpis składa się z selektora, do którego planujemy przypisać identyfikator, oraz z
nazwy poprzedzonej znakiem #.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. Identyfikator może wystąpić tylko raz w całym dokumencie.
  2. Podobnie jak klasy, identyfikatorów, możemy używać tyle razy, ile razy uznamy to za
    konieczne. Działanie takie jest błędem, ale nie wpływa na poprawność działania strony.
  3. Elementy oznaczone ID muszą być unikatowe czyli może wystąpić tylko jeden element z takim ID.
